Detailed Description

Protótipos do Real-Time Database (RTDB) para o controlador térmico.

Este ficheiro define a estrutura rtdb_t e as funções de acesso (getters/setters) protegidas por mutex, de modo a permitir comunicação segura entre várias tasks.
