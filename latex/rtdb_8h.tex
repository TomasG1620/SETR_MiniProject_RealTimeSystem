\doxysubsection{Detailed Description}
Protótipos do Real-\/\+Time Database (RTDB) para o controlador térmico. 

Este ficheiro define a estrutura \doxylink{structrtdb__t}{rtdb\+\_\+t} e as funções de acesso (getters/setters) protegidas por mutex, de modo a permitir comunicação segura entre várias tasks.
