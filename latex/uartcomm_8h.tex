\doxysubsection{Detailed Description}
Interface do módulo de comunicação UART (parser de comandos + framing) 

Este header exporta apenas a função \doxylink{uartcomm_8h_a831861af1bf5e28dca2ae8b1b7f27b3e}{uart\+\_\+comm\+\_\+init()}, que inicia uma thread responsável por ler bytes da UART em modo polling, reconstituir frames do tipo “\#$<$\+CMD$>$$<$\+DATA$>$$<$\+CS$>$!” e disparar o tratamento de cada comando.
